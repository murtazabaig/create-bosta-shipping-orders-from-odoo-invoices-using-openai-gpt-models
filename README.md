# Create Bosta shipping orders from Odoo invoices using OpenAI GPT models

Advanced n8n automation for Create Bosta shipping orders from Odoo invoices using OpenAI GPT models.

## Overview
- Category: CRM, AI RAG
- Complexity: advanced
- Source: n8n workflow template export

## What This Automation Does
Automate OdooBosta shipping: auto-create orders on invoice confirmation, AI-mapped Zone/District IDs, clean labels, COD rounding, Telegram alerts.

## Included Files
- `workflow.json`
- `metadata.json`

## Setup
1. Import `workflow.json` into n8n.
2. Configure required credentials for the services used in the workflow nodes.
3. Update any environment variables or static values inside nodes (API keys, URLs, IDs).
4. Run a test execution and then activate the workflow.
